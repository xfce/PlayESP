# PlayESP
Play with ESPEASY

## PlayESP_V1.0
### ESPHOME _V1.0硬件特性

1）板载USB转串口及固件升级按键；

2）BMP280，MS5611气压传感器；

3）MPU6050三轴加速度传感器；

4）SHT10温湿度传感器；

5）板载锂电池充电电路及电池电量检测，支持外接锂电池供电；

6）一个TSOP4838红外解码学习；

7）7个TSAL6200全方位红外发射管；

8）ESP8266可用IO口全部引出，方便扩展。
